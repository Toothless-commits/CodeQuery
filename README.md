
# CodeQuery

CodeQuery is a simple Question & Answer web application built with Flask and PostgreSQL. Users can register, log in, ask questions, post answers, search existing discussions, and manage their content through an easy-to-use interface.

## Features

- User registration and login
- Secure password hashing using bcrypt
- Ask questions
- Post answers
- Search questions and answers
- Edit and delete questions
- User profile showing posted questions and answers
- PostgreSQL database integration

## Tech Stack

- Python
- Flask
- PostgreSQL
- HTML
- CSS
- bcrypt
